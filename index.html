Help Menu

 
 

The MPN package computes the Most Probable Number, i.e. microbial density, derived from serial dilutions.

The user inputs the number of dilutions, number of tubes, number of positive tubes, amount of inocula.

 
 

+


This button will add a new set to the data set that includes: Positive, Population, Volume.

 

Positive:  Number of Positive tubes.

Population: Total number of tubes.

Volume: Total quantity of dilution being used in each tube.

 

Query

Calculates data set using the MPN formula to find the most probable number.

 
 

If NaN received as an answer: NaN represents "Not-A-Number"value.
